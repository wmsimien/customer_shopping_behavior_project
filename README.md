# Customer Behavior Dashboard

_This project represens a complete end-to-end data analytics workflow which encompasses all critical stages of analysis, preparation and modeling, visualization and reporting_

## 🎯 Project Overview

This project simulates the end-to-end data analytics workflow of cleaning and transforming raw cutomer data, extract insigts on cutomer segment, loyalty, and purchase drivers.  An interactive dashborad is built highlighting key patterns and trends enabling stakeholders to make data-driven decisions.  

## 🛠️ How to Use This Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/wmsimien/customer_shopping_behavior_project.git
   cd customer_shopping_behavior_project
   ```
2. **Open Customer_Shopping_Behavior_Analysis.ipynb notebook**

    This file contains:

      - Data Import

      - Data exploration

      - Data cleaning

      - Connection to PostreSQL Database
  
3. **Load the data from Python notebook into PostgreSQL Server**

      - Create a database in PostgreSQL

      - Run Python code to load data into database
  
      - Open **customer_shopping_behavior_analysis_queries.sql**
  
      - Answer Business Questions using SQL Queries 
      
4. **Connect the PostgreSQL Database to Power BI**

      - Open **customer_shopping_behavior_dashboard.pbix**
   
      - Create interactive dashboard in Power BI
